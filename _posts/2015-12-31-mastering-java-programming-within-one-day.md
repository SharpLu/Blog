1	java语言简介
1.1 	java基本概念
JDK：Java Development Kit （ Java 开发必备）
JRE：Java Runtime Environment （ Java 执行环境）
1.2 	Java具有平台无关性
Java是一种既面向对象又可以跨平台的语言，即：编写一次，随处运行；
Java不仅适用于单机应用程序和基于网络的程序，也可用于创建消费类设备的附件程序，如移动电话、掌上设备等。
